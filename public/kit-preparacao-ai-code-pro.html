Bem-vindo ao Futuro da IA!

Para dominar IA de verdade no nosso evento de 01 a 04 de setembro (20h às 22h), você precisa preparar seu ambiente de desenvolvimento. Este guia contém tudo que você precisa!

Requisitos Mínimos do Sistema

  • Windows 10/11, macOS 10.15+ ou Linux Ubuntu 18.04+
  • 8GB de RAM (recomendado: 16GB para melhor performance)
  • 10GB de espaço livre em disco
  • Conexão estável com a internet (para downloads e IA)

Ferramentas Essenciais

Python 3.11+
Linguagem principal que usaremos para desenvolver aplicações de IA, trabalhar com LLMs, criar agentes inteligentes e implementar RAG (Retrieval-Augmented Generation).
Baixar Python

Passos de Instalação:

1 Baixe a versão mais recente do Python (3.11 ou superior)
2 ✅ IMPORTANTE: Marque "Add Python to PATH" durante a instalação
3 Teste no terminal/prompt: python --version
Node.js 18+
Necessário para ferramentas de desenvolvimento, gerenciadores de pacotes e algumas bibliotecas de IA que usaremos durante o evento.
Baixar Node.js

Passos de Instalação:

1 Baixe a versão LTS (Long Term Support) - mais estável
2 Execute o instalador com configurações padrão
3 Teste no terminal: node --version e npm --version
Visual Studio Code
Editor de código principal que usaremos. Leve, poderoso e com excelente suporte para Python, IA e desenvolvimento em geral.
Baixar VS Code

Extensões Recomendadas:

1 Python (Microsoft) - Suporte completo para Python
2 Pylance - IntelliSense avançado para Python
3 Jupyter - Para notebooks interativos
4 GitHub Copilot (opcional) - Assistente de IA para código
Windsurf (Codeium)
Nossa arma secreta! Editor de código revolucionário com IA integrada. Ferramenta que usaremos para desenvolvimento assistido por IA de última geração.
Baixar Windsurf

Configuração:

1 Crie uma conta gratuita no Codeium
2 Baixe e instale o Windsurf
3 Configure a integração com IA
4 Teste o assistente de código - você vai se surpreender!

Dicas Importantes

Tenha um backup: Se algo não funcionar no seu computador, tenha o Google Colab como alternativa.
Pratique com Windsurf: É nossa ferramenta secreta - quanto mais você usar antes do evento, melhor será sua experiência!
Entre no grupo do WhatsApp: Qualquer dúvida sobre instalação, pergunte no grupo exclusivo!
